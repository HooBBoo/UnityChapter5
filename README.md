공동묘지 만들기

## ⚙ 주요기능

- 마우스가 바라보는 곳의 정보가 표시됩니다. (묘비의 비문 표시)

- [E]로 상호작용하면 도굴과 씨앗찾기가 가능합니다.

- [R]로 튜터님 동상과 상호작용하면 정신건강이 회복됩니다.

- 근처에 있으면 정신건강수치가 줄어드는 NPC가 맵을 돌아다닙니다.
